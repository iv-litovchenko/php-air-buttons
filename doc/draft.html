5. Как использовать эту схему

Этот файл не подключён к TYPO3 и ничего не знает о реальных путях/BE‑модулях — он нужен как “визуальная шпаргалка” для восстановления логики, если проект выносится в отдельное приложение или переписывается UI.

При переносе в самостоятельный проект можно:

  • сохранить идею маркеров <!--@@@...@@@--> в HTML,
  • переписать post‑processor (middleware), который будет их заменять,
  • перенастроить целевые URL/действия кнопок на новый backend/API.

PHP — fluent-интерфейс (спецификация)

Целевой способ вызова верхней панели из шаблона: цепочка настроек и ->render(). Здесь описан только API; реализация класса может быть добавлена отдельно. 

<?php
// после подключения вашего класса PHPAirButton:

PhpAirButton::adminPanelCreate()
    ->brand('SPRAVKY.RU', '/')
    ->user($currentUserName ?? 'Editor')
    ->pageId((string)($currentPageId ?? 0))
    ->editing($isEditingOn ?? true)
    ->backend('/admin/')
    ->logout('/logout')
    ->render();
?>

Минимальный набор для работающей панели: brand(), user(), pageId(), editing(), backend(), logout(). Опционально можно добавить: clearCache(), colors(), settings(), materialsRows([...]) и materialsTotal(...). 3) Панели EditIcon (современный стиль)

EditIcon — существующий блок

Наведи курсор на плашку “Новость” (иконка + текст) — справа от неё появятся кнопки редактирования.

Точка входа: PhpAirButton::editIcons() (отдельно от верхней панели). existing($type, $id, $chipUrl, $editUrl): $chipUrl — ссылка плашки (иконка + «Новость №…»), $editUrl — кнопка ✎. Дополнительно цепочкой: hideUrl($url), deleteUrl($url), createSiblingUrl($url), tooltip($text) (и при необходимости статус записи отдельным методом билдера). 

<?php
PhpAirButton::editIcons()
    ->existing('Новость', '1993', '#record-news-1993', '#edit-record-1993')
    ->hideUrl('#hide-record-1993')
    ->deleteUrl('#delete-record-1993')
    ->tooltip('Тип записи / заголовок элемента')
    ->render();
?>
Текст новости №1993
Здесь какой‑то текст записи. Панель действий привязана к плашке “Новость №1993” ниже.
▤ Новость №1993
✎ ☼ ✖

EditIcon — новый блок (placeholder)

＋Create here
Панель получает параметры размещения и открывает мастер создания записи.

Точка входа: PhpAirButton::newIcon(). placeholder($type, $createUrl, ?$label): ссылка на создание и подпись на плашке (как «Create here»). Дополнительно: tooltip($text) (атрибут title), icon($glyph), disabled($on, $reason). 

<?php
PhpAirButton::newIcon()
    ->placeholder('Новость', '#new-tt-content-here', 'Create here')
    ->tooltip('Создать контент в этой колонке')
    ->render();
?>
